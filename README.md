# Yordanos Haliso Hanjalo — Portfolio

A premium dark-mode portfolio website built with **React + Vite**, featuring glassmorphism UI, neon accents, animated data visualizations, and smooth scroll animations.

## ✨ Features

- **Dark Mode** — True black backgrounds with Electric Blue & Magenta neon accents
- **Glassmorphism** — Frosted glass cards with backdrop blur effects
- **Animated KPIs** — SVG ring charts and counter animations for marketing metrics
- **Interactive Projects** — Hover-reveal KPI overlays on project cards
- **Experience Timeline** — Vertical timeline with glowing neon nodes
- **Vibe Coder Section** — AI-assisted development as a unique selling point
- **Responsive** — Mobile-first design with smooth transitions
- **SEO Optimized** — Meta tags, Open Graph, semantic HTML

## 🛠 Tech Stack

- React 19
- Vite
- Framer Motion
- React Icons
- React Intersection Observer

## 🚀 Getting Started

```bash
npm install
npm run dev
```
